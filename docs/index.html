// The problem

Every app you use is built on code written by strangers.

Modern software doesn't get built from scratch. It's assembled from thousands of open source packages — small pieces of code written by people all over the world. Your banking app, your medical records portal, your kid's school platform — they all depend on this invisible supply chain.

That's a problem. Because any one of those packages can be hijacked, abandoned, or written with hidden backdoors. When that happens, attackers don't break in through the front door — they come in through the building materials.

Thresher is a free, open source tool that scans those building materials — every dependency, every layer — and tells you what's safe and what's not. One command, full picture.

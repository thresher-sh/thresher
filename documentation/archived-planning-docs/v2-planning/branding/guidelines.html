thresher brand guidelines

Separate the safe from the dangerous.

// Identity

Thresher is a supply chain security scanner. The name references the threshing machine (separating grain from chaff), the thresher shark (fast, precise, relentless), and the .sh domain (shell-native).

Install with one command:
curl -fsSL thresher.sh/install | bash

// Color Palette

Dark-themed, terminal-native. Shark blue is the single accent color. Red, green, and amber are functional only (severity indicators), never decorative.

Primary

Deep Black#0a0a0a — backgrounds
Terminal Black#1a1a1a — surfaces, cards
Charcoal#2a2a2a — borders
Steel#6b7280 — secondary text
Bone#e5e5e5 — body text
White#fafafa — headings

Accent

Shark Blue#38bdf8 — accent, links, brand

Functional (severity only)

Threat Red#ef4444 — critical, errors
Amber#f59e0b — medium, caution
Safe Green#22c55e — clean, pass

// Typography

thresher scan https://github.com/...

// Voice & Tone

Direct. Technical. Confident. Terse. Assume the reader knows what a CVE is.

Do
Thresher scans your dependencies for threats.
Don't
Thresher empowers developers to proactively manage supply chain risk in their software ecosystem.
Do
8 analysts. Every angle. Nothing hides.
Don't
Our comprehensive multi-agent AI architecture provides thorough coverage across multiple security domains.
Do
Found 2 critical, 5 high. Top risk: CVE-2026-1234 in lodash. Upgrade to 4.17.21.
Don't
We've identified several potential security concerns that may require your attention at your earliest convenience.

// Usage Rules

Do

Use lowercase thresher in text and CLI context

Use Shark Blue as the single accent color

Use monospace as the primary typeface

Keep dark backgrounds for all brand materials

Use the tail fin as the favicon/icon

Don't

Don't capitalize as "Thresher" in CLI output

Don't use multiple accent colors

Don't use sans-serif as the primary typeface

Don't create light-mode-first materials

Don't use the shark as a cartoon/mascot character

# Report

There are several reports that you should take note of.

## Analysts

Every analyst outputs its own markdown report. Look if you want, but they are rolled up into others.

## Adversarial

It shows what it debunked or changed.

## Synthesis Reports

These are important...

The `detailed-report.md` and `executive-summary.md` are the most important files. They have all the details and outcomes from the run.

## HTML Report

The html report is an end user friendly report that is ready to take and show to a manager, leader, etc... It encompasses all the details from detailed-report and executive summary.
